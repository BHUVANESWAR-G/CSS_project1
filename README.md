# 🌸 Perfumy – Perfume Website

A simple and elegant perfume showcase website built using **HTML** and **CSS**. This project displays different perfume products with a clean UI and basic layout structure.

---

## 📸 Preview

* Homepage with navigation bar
* Product showcase section
* About section
* Contact section

---

## 🚀 Features

* 🧭 Navigation bar (Home, Products, About, Contact)
* 🔍 Search bar UI
* 🛍️ Product display with images
* 📄 About section for brand description
* 📞 Contact information section
* 🎨 Hover effects for better user experience
* 📱 Responsive-centered layout (basic)

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website 
* **CSS3** – Styling and layout design 

---

## 📂 Project Structure

```
Perfumy/
│── index.html
│── style.css
│── img/
│   ├── Perfume1.jpg
│   ├── Perfume2.jpg
│   └── Perfume3.jpg
```

---

## 📖 How It Works

* The **navigation bar** provides basic page sections.
* The **search bar** allows users to type queries (UI only).
* The **products section** displays perfume images with descriptions.
* The **about section** explains the brand.
* The **contact section** shows contact details.

---

## ▶️ How to Run

1. Download or clone the repository
2. Make sure the folder structure is correct (`img` folder included)
3. Open `index.html` in your browser
